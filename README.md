# Llama 3.2 instruct for indic languages
This project studies PEFT methods -LoRA and QLoRA on LLaMA 3.2 for multilingual Indic instruction tuning tasks such as translation, paraphrasing, and summarization. Samanantar dataset is converted to structured instruction–response pairs. We study trade-offs between accuracy and computational efficiency by comparing fine-tuning with LoRA and QLoRA, with metrics-BLEU/ROUGE scores, GPU memory usage, and training time


Model:
https://huggingface.co/meta-llama/Llama-3.2-3B

Datasets: 
https://huggingface.co/datasets/ai4bharat/samanantar
